# Verifica di Programmazione Python
## Consegna
Consegna tre file Python distinti, rinominati come `cognome1.py`, `cognome2.py`, `cognome3.py` (es. `rossi1.py`, `rossi2.py`, `rossi3.py`).  
Testa il codice eseguendolo in terminale con `python nomefile.py`.

## Istruzioni Generali
- Scrivi codice Python 3.x pulito, con commenti concisi.
- Gestisci errori di input (es. valori negativi o invalidi) stampando messaggi appropriati.

---

## Esercizio 1: Cicli Senza Liste con Calcoli
Scrivi un programma che chieda quanti numeri inserire (n > 0), li legga uno per uno, calcoli la somma dei quadrati dei numeri pari, il conteggio dei numeri dispari negativi e la media assoluta (valore assoluto) di tutti i numeri.  
Gestisci n <= 0 con errore. Usa cicli `for` senza liste.

Nota: Utilizza la funzione `abs()` per calcolare il valore assoluto dei numeri. Esempio: `abs(-5)` restituisce `5`.

### Esempio
**Input:**  
n = 5, numeri: 2, -3, 4, -5, 6  

**Output:**  
Somma quadrati pari: 40 (2² + 4² + 6²)  
Conteggio dispari negativi: 2  
Media assoluta: 4.0 ((|2| + |-3| + |4| + |-5| + |6|) / 5)

---

## Esercizio 2: Liste con Filtri e Statistiche
Scrivi un programma che chieda quanti numeri interi inserire (n > 0), li legga in una lista, poi crei una nuova lista con solo quelli multipli di 3 e positivi, conti quanti sono nella nuova lista, quanti sono nella lista originale ma non nella nuova, e calcola la somma dei numeri nella nuova lista.  
Gestisci n <= 0 con errore. Stampa la lista filtrata, il conteggio filtrati, il conteggio non filtrati e la somma.

### Esempio
**Input:**  
n = 6, numeri: 3, 6, -9, 12, 5, 15  

**Output:**  
Lista filtrata: [3, 6, 12, 15]  
Filtrati: 4  
Non filtrati: 2  
Somma filtrati: 36

## Esercizio 3: Liste Parallele con Calcoli
Scrivi un programma che definisca tre liste parallele: `prodotti` (string, nomi prodotti), `prezzi` (float, prezzo unitario), `quantita` (int, quantità disponibile).  

- Calcola il valore totale (prezzi * quantita) per ogni prodotto in una lista `valori`.  
- Trova il prodotto con valore totale massimo.  
- Conta quanti hanno **prezzo < 10 AND quantità > 5**.  

Verifica che le liste abbiano la stessa lunghezza. Usa liste predefinite come ["Pane", "Latte", "Uova"], [2.5, 1.2, 12.0], [10, 20, 15].  
Stampa i prodotti con valori, il prodotto con valore massimo e il conteggio.

### Esempio
**Output:**  
```
Prodotti:  
Pane: 25.0 €  
Latte: 24.0 €  
Uova: 180.0 €  
Prodotto con valore massimo: Uova  
Prodotti con prezzo <10 e quantità >5: 2  (solo Pane e Latte soddisfano entrambe le condizioni; Uova ha prezzo 12.0 >=10)
```
