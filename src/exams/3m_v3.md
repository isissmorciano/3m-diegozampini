# Verifica su Funzioni - 2 Esercizi

## Esercizio 1: Funzione per Validazione Password

**Obiettivo:** Creare funzioni per verificare password (lunghezza >=8, almeno 1 maiuscola, 1 minuscola, 1 numero, prima lettera maiuscola, non in lista password comuni).

**Istruzioni:**
1. Definisci lista globale `PASSWORD_COMUNI = ["password", "123456", "qwerty", "admin", "letmein"]`.
2. Definisci `conta_caratteri(password: str) -> dict[str, int]` che restituisce un dizionario con chiavi "maiuscole", "minuscole", "numeri" e valori interi corrispondenti al conteggio (es. {"maiuscole": 1, "minuscole": 5, "numeri": 3} per "MyPass123").
3. Definisci `valida_password(password: str) -> bool` che restituisce True se tutti i requisiti sono soddisfatti, False altrimenti.
4. Definisci `livello_sicurezza(password: str) -> int` che restituisce un intero da 1 a 3: inizia con livello 1, aggiungi +1 se lunghezza >=10, aggiungi +1 se maiuscole >1 OR numeri >1.
5. Nel main, chiedi password all'utente, chiama valida_password e livello_sicurezza, stampa "Valida (Livello X)" se valida, "Non valida" altrimenti.

**Esempio di Utilizzo:**
- Avvio programma: "Inserisci password: MyPass123"
- Output: "Valida (Livello 2)"
- Spiegazione: Lunghezza 9 (>=8), prima M maiuscola, 2 maiuscole (M, P), 4 minuscole, 3 numeri, non comune → valida, livello 2 (lunghezza >=10? No, ma ha >1 maiuscola e >1 numero).

**Esempio:**
- Input: "MyPass123"
- Output: Valida (Livello 2)

**Suggerimenti:**
- Una stringa può essere interpretata come una lista: puoi fare un ciclo su di essa con `for char in password` per iterare su ogni carattere.
- Usa `char.isupper()` per verificare se un carattere è maiuscolo.
- Usa `char.islower()` per verificare se un carattere è minuscolo.
- Usa `char.isdigit()` per verificare se un carattere è una cifra numerica.
- Usa `password.lower()` per convertire la password in minuscoli e confrontarla con le password comuni.

## Esercizio 2: Gestore Ricette con Approccio Top-Down

**Obiettivo:** Creare un programma per gestire ricette, aggiungendo ingredienti con quantità e prezzo, calcolando costo totale. Usa approccio top-down con funzioni separate.

**Istruzioni:**
1. Definisci `aggiungi_ingrediente() -> dict[str, float | str]` che chiede nome, quantità (float), prezzo unitario (float) all'utente e **restituisce un dizionario** {"nome": str, "quantita": float, "prezzo_unitario": float} (es. {"nome": "farina", "quantita": 500.0, "prezzo_unitario": 0.01}).
2. Definisci `calcola_costo_totale(ricetta: list[dict[str, float]]) -> float` che riceve la lista ricetta e restituisce un float con la somma di quantita * prezzo_unitario per ogni ingrediente (es. 5.0 per [{"nome": "farina", "quantita": 500, "prezzo_unitario": 0.01}]).
3. Definisci `visualizza_ricetta(ricetta: list[dict[str, float]]) -> None` che riceve la lista ricetta, stampa ogni ingrediente con formato "nome: quantita x prezzo_unitario = costo" (es. "farina: 500 x 0.01 = 5.0"), e alla fine "Totale: X €". Non restituisce nulla.
4. Definisci `main() -> None` che inizializza ricetta come lista vuota, mostra un menu ciclico (1. Aggiungi, 2. Visualizza, 3. Totale, 4. Esci), chiama le funzioni appropriate in base alla scelta. Per aggiungere: `ricetta.append(aggiungi_ingrediente())`. Termina al 4.
5. **Struttura dati ricetta**: Rappresenta la ricetta come una **lista di dizionari**. La lista **inizia vuota** (`ricetta = []`) nel main e si popola man mano che l'utente aggiunge ingredienti. Ogni elemento della lista è un ingrediente (dizionario con chiavi "nome", "quantita", "prezzo_unitario"). Quando l'utente sceglie "Aggiungi", la funzione `aggiungi_ingrediente()` restituisce un nuovo dizionario che viene aggiunto alla lista tramite `ricetta.append()`. Esempio finale: `ricetta = [{"nome": "farina", "quantita": 500.0, "prezzo_unitario": 0.01}, {"nome": "uova", "quantita": 2.0, "prezzo_unitario": 0.3}]`.

**Esempio di Utilizzo:**
- Avvio: Menu appare.
- Scelta 1: "Nome ingrediente: Farina", "Quantità: 500", "Prezzo unitario: 0.01" → aggiunto.
- Scelta 1: "Nome ingrediente: Uova", "Quantità: 2", "Prezzo unitario: 0.3" → aggiunto.
- Scelta 2: Output:
  Farina: 500 x 0.01 = 5.0
  Uova: 2 x 0.3 = 0.6
  Totale: 5.6 €
- Scelta 3: "Totale: 5.6 €"
- Scelta 4: Esce.

**Esempio:**
- Aggiungi: Farina, 500, 0.01
- Totale: 5.0 €
